# Statsez API — LLM Reference

> Bilingual (English / Português). [EN] = English. [PT] = Português.

---

## Overview / Visão Geral

[EN] Statsez API provides **historical football data** from 750+ worldwide leagues.
It does NOT provide real-time livescores. All data is post-match.

[PT] A Statsez API fornece **dados históricos de futebol** de 750+ ligas mundiais.
NÃO fornece livescore. Todos os dados são pós-partida.

---

## Base Configuration / Configuração

- **Base URL:** `https://api.statsez.com/v1`
- **Auth:** Header `x-api-key: YOUR_KEY`
- **Format:** JSON (UTF-8)
- **Rate Limit:** 100 req/min per IP
- **Quota:** Bi-weekly cycle (15 days), resets automatically
- **Quota Headers:** `X-RateLimit-Limit`, `X-RateLimit-Remaining`, `X-RateLimit-Reset`

## Endpoints (11)

### 1. LIST LEAGUES
`GET /football/leagues`
- Params: `country` (optional)
- Returns all competitions

```bash
curl -H "x-api-key: YOUR_KEY" "https://api.statsez.com/v1/football/leagues?country=Brazil"
```
```json
{
  "success": true,
  "data": {
    "leagues": [
      { "id": "/football/brazil/serie-a/", "name": "Série A", "country": "Brazil", "season": "", "slug": "/football/brazil/serie-a/" }
    ],
    "total": 14
  },
  "meta": { "cached": true, "remainingQuota": 480 }
}
```

### 4. STANDINGS
`GET /football/standings`
- Params: `league` (**required** — tournament_stage_id), `season` (optional)

```bash
curl -H "x-api-key: YOUR_KEY" "https://api.statsez.com/v1/football/standings?league=dINOZk9Q"
```
```json
{
  "success": true,
  "data": {
    "league": "LaLiga",
    "country": "Spain",
    "season": "",
    "standings": [
      { "position": 1, "team": "Barcelona", "teamId": "SKbpVP5K", "played": 38, "won": 28, "drawn": 4, "lost": 6, "goalsFor": 102, "goalsAgainst": 39, "goalDifference": 63, "points": 88 },
      { "position": 2, "team": "Real Madrid", "teamId": "W8mj7MDD", "played": 38, "won": 26, "drawn": 6, "lost": 6, "goalsFor": 78, "goalsAgainst": 38, "goalDifference": 40, "points": 84 }
    ]
  }
}
```

> **Note:** For standings, the `league` param is the `tournament_stage_id` (e.g. `dINOZk9Q` for LaLiga 2024-25). You also need to pass the `tournament_id` internally. Use `/matches/details` on any match to discover these IDs.

## Quick Start (Python)

```python
import requests

KEY = "se_live_your_key_here"
BASE = "https://api.statsez.com/v1"
H = {"x-api-key": KEY}

# List leagues
leagues = requests.get(f"{BASE}/football/leagues?country=England", headers=H).json()

# Get match details + stats
match = requests.get(f"{BASE}/football/fixtures/ngWM9fgd", headers=H).json()
print(match["data"]["fixture"]["homeTeam"]["name"])  # "Leeds"
print(match["data"]["stats"]["Ball possession"])      # ["38%", "62%"]

# Get standings (needs tournament_stage_id)
standings = requests.get(f"{BASE}/football/standings?league=dINOZk9Q", headers=H).json()
```
